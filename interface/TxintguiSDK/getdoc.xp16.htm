François Guibert wrote in message ...
>Hi!
>
>I'd like to know if there is a way to change the color of a static control.
>In fact, I want to write in white ON black instead of black on grey.
  
 
Certainly, yes. Respond to the WM_CTLCOLORSTATIC message in the parent of the control (ie in the dialog). In the message handler, call "SetTextColor" to set the text colour, and return from the handler the handle of the brush to be used to draw the control's background.
