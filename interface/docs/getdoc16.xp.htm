Send a CB_SETITEMDATA message to the combo, passing in the array index. Then when you need to access the array, send a CB_GETITEMDATA message, passing in the selected item index -- it'll return the array index.
 
Or you can turn off CBS_SORT and leave the combo unsorted, if that's acceptable.
