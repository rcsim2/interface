>Could someone please explain in detail the difference between
>VPICD_Phys_EOI(hIRQ), and VPICD_Physically_Unmask(hIRQ).
  
VPICD_Physically_Unmask allows interrupts to start happening.  When an interrupt is masked, all interrupts for that IRQ are suppressed until it is unmasked.  Unmask is typically done once during initialization.
 
When an interrupt arrives, all further interrupts on that IRQ are suspended until the processor asserts that it has completed processing and is ready to handle another one.  VPICD_Phys_EOI does that.
 
Thus, Physically_Unmask is a global on/off switch for the IRQ.  Phys_EOI is done at the end of EVERY interrupt you handle.
