Hi,
You can try this:
 
VPICD_Set_Int_Request(hVM, hIRQ);
VPICD_Clear_Int_Request(hVM, hIRQ);
VPICD_Phys_EOI(hIRQ);
 
This sequence at the end of my interrupt handler woks fine...
 
Laurent Mealares.
