In article <38396995@es.classicweb.com>, "AMP" <tu__la@teleline.es> wrote:
>
>I used ADRC2VXD to add version info to
>my VxD (ADRC2VXD myvxd.vxd myresfile.res)
>and it returns with the following error
>message:
>
>"Error: while finding version info in
>resource file"
>
>To compile the .rc file I used the resource
>compiler which comes with the win98DDK.
>
>Any idea?
  
You have used the 32bit resource compiler in <DDKPATH>\bin\rc32.exe You need to use the 16bit compiler in <DDKPATH>\bin\win98\bin16\rc16.exe
 
After you compile the resources to te correct binary format then ADDRC2VXD should work correctly.
